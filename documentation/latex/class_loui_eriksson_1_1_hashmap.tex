Version 1.\+0.\+1 

Custom \mbox{\hyperlink{class_loui_eriksson_1_1_hashmap}{Hashmap}} implementation accepting a customisable key and value type. Created using a combination of prior knowledge and brief online tutorial. 

This implementation requires that your \char`\"{}key\char`\"{} type is compatible with std\+::hash and that the stored data types are copyable.

Reference\+: Wang, Q. (Harry) (2020). Implementing Your Own Hash\+Map (Explanation + Code). You\+Tube. Available at\+: \href{https://www.youtube.com/watch?v=_Q-eNqTOxlE}{\texttt{ https\+://www.\+youtube.\+com/watch?v=\+\_\+\+Q-\/e\+Nq\+TOxlE}} \mbox{[}Accessed 2021\mbox{]}.
