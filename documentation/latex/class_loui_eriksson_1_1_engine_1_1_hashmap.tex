Custom \mbox{\hyperlink{class_loui_eriksson_1_1_engine_1_1_hashmap}{Hashmap}} implementation accepting a customisable key and value type. Created using a combination of prior knowledge and brief online tutorial. 

This implementation requires that your \char`\"{}key\char`\"{} type is compatible with std\+::hash and that the stored data types are copyable.
